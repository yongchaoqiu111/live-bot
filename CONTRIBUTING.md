# 贡献指南

感谢你考虑为本项目做出贡献！

## 如何贡献

### 1. Fork项目
点击右上角的"Fork"按钮

### 2. 创建分支
```bash
git checkout -b feature/your-feature-name
```

### 3. 提交修改
```bash
git add .
git commit -m "feat: 添加新功能"
```

### 4. 推送并创建Pull Request
```bash
git push origin feature/your-feature-name
```

## 开发规范

### 代码风格
- 遵循PEP 8规范
- 使用中文注释
- 函数需有文档字符串

### 提交信息格式
- `feat:` 新功能
- `fix:` 修复bug
- `docs:` 文档更新
- `refactor:` 代码重构
