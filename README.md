# 直播竞品复刻实战系统

> 🎯 **四维成交线索闭环：话术+弹幕+商品+销量，看透头部主播的成交逻辑**

---

## 📖 项目简介

这不是一个技术炫耀项目，而是一套**AI时代普通人能落地的直播竞品分析方案**。

### 💡 核心价值

**别人的工具告诉你"谁卖得好"，我们告诉你"他怎么做到了"**

- ❌ 传统工具：只给GMV、在线人数等结果数据
- ✅ 本系统：**话术+弹幕+商品+销量**四维闭环，精确归因到每一句话

### 🎯 适用人群

- **中小商家**：想抄头部主播作业，不知道从哪下手
- **直播运营**：想优化话术节奏，没有数据依据  
- **MCN机构**：想批量复制成功直播间，缺方法论
- **个体创业者**：想低成本入局直播，不想瞎摸索

### 🔥 核心优势

1. **四维数据闭环**：话术（过程）+弹幕（反馈）+商品（载体）+销量（结果）
2. **销量归因能力**：把销量绑定到每一句话，找出真正的"黄金话术"
3. **LLM智能分析**：从数据到结论，自动生成可执行策略
4. **低风控低成本**：录音+低频抓取，个人即可操作

## 🏗️ 技术架构

### 完整技术链路

```
录音 → Whisper转文字 → 话术原文
弹幕抓取 → 时序存储 → 弹幕原文  
商品抓取 → 时序存储 → 商品动作
销量抓取 → 时序存储 → 销量曲线
                ↓
         统一时间轴对齐
                ↓
         【LLM分析层】← 核心
                ↓
    结构化清洗 → 关联归因 → 策略生成
                ↓
         最终报告（人可直接用）
```

### 技术栈

| 模块 | 技术选型 | 说明 |
|------|---------|------|
| **开发语言** | Python 3.10+ | 主流AI开发语言 |
| **浏览器自动化** | Playwright | 弹幕/商品/销量抓取 |
| **音频录制** | PyAudio + pydub | 系统声卡录制，60秒切片转mp3 |
| **语音识别** | Whisper-large-v3 | OpenAI开源模型，离线转文字 |
| **LLM分析** | GPT-4o / Claude 3.5 | 话术分类、销量归因、策略生成 |
| **数据存储** | SQLite | 本地数据库，零部署成本 |
| **可视化** | Matplotlib | 销量曲线、弹幕热度图 |
| **报告生成** | Markdown + Excel | 标准化竞品分析报告 |

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
playwright install chromium
```

### 2. 配置API密钥

编辑 `config/settings.py`：

```python
OPENAI_API_KEY = "sk-your-api-key"  # 用于LLM分析
```

### 3. 运行系统

```bash
python main.py
```

输入直播间URL，系统自动：
1. ✅ 录音（60秒切片，转mp3）
2. ✅ 抓取弹幕（5秒间隔）
3. ✅ 抓取商品+销量（30秒间隔）
4. ⏸️ 按Ctrl+C停止
5. 🔄 Whisper转文字
6. 🤖 LLM三层分析（清洗→归因→策略）
7. 📊 生成可视化图表
8. 📄 输出15-20页专业报告

## 📊 输出指标体系

### 1️⃣ 话术结构指标
- 憋单/逼单/卖点/互动话术分布
- 平均憋单时长、逼单频率
- 黄金话术模板（可直接复制）

### 2️⃣ 直播节奏指标  
- 开播前10分钟留人策略
- 上新品时间间隔
- 逼单节奏分布
- 分钟级节奏脚本

### 3️⃣ 商品效率分析
- 主推商品TOP3（按销量效率排序）
- 每款商品讲解时长vs销量
- 商品排品建议

### 4️⃣ 用户需求分析（弹幕）
- 高频问题TOP10
- 情绪分布（急切购买/犹豫观望/质疑负面）
- 核心痛点提取
- 弹幕活跃度曲线

### 5️⃣ **销量归因分析（核心）**
- 高转化时刻归因（TOP3）
  - 销量增量多少
  - 主播说了什么话术
  - 弹幕反应如何
  - 当时讲解什么商品
  - 为什么卖得好
  
- 低效时刻诊断（TOP3）
  - 问题出在哪里
  - 如何优化

### 6️⃣ 可复制策略
- 3条高转化话术模板（带预期效果）
- 直播节奏优化建议
- 商品排品建议
- 避坑指南

## 🔧 核心功能详解

### 四维数据闭环

#### 1. 话术维度
- 录音留存完整直播音频
- Whisper离线转文字，标注时间节点
- LLM分类：憋单型/逼单型/卖点型/互动型/信任型

#### 2. 弹幕维度
- 实时抓取弹幕文本
- LLM清洗：提取高频词、情绪分布、核心痛点
- 峰值时刻识别

#### 3. 商品维度
- 抓取商品上架信息
- 记录讲解时段
- 价格变动追踪

#### 4. **销量维度（核心创新）**
- 每30秒抓取商品卡片"已售"数量
- 自动计算销量增量
- 销量爆发时间点定位
- **销量归因**：将销量绑定到具体话术+商品+弹幕

### LLM分析层（三层递进）

#### 第一层：结构化清洗
- Prompt 1：话术结构化（提取类型、卖点、行动指令）
- Prompt 2：弹幕清洗（提取高频词、情绪、痛点）

#### 第二层：关联归因
- Prompt 3：销量归因分析
  - 找出高转化时刻
  - 诊断低效时刻
  - 输出归因结论

#### 第三层：策略生成
- 高转化话术模板（3条）
- 直播节奏优化建议
- 商品排品建议
- 一句话核心结论

## ❓ 常见问题

**Q：会不会被封号？**  
A：只用录音+低频抓取（5-30秒一次），模拟正常人浏览，不破解接口，风控风险极低。

**Q：需要多长时间？**  
A：一场3小时直播，抓取约2-4小时，LLM分析约5-10分钟。

**Q：支持哪些平台？**  
A：抖音、快手、淘宝等主流平台，只需调整DOM选择器。

**Q：分析结果准确吗？**  
A：销量归因基于数据统计，建议多次测试取规律，而非单次下结论。
